# Agenda en Python (sencilla)

Proyecto de una agenda de contactos desarrollada en Python utilizando
Programación Orientada a Objetos (POO).

El objetivo del proyecto es practicar el uso de clases, métodos,
encapsulamiento y manejo básico de datos desde consola.

---

## Funcionalidades

- Agregar contactos
- Eliminar contactos
- Ver la lista de contactos
- Menú interactivo por consola

---

## Estructura del proyecto

- `persona.py` → Clase Persona (datos del contacto y validaciones)
- `menu.py` → Lógica del menú y operaciones
- `main.py` → Punto de entrada del programa

---

## Cómo ejecutar el programa

1. Clonar el repositorio
2. Abrir una terminal en la carpeta del proyecto
3. Ejecutar el archivo principal:

```bash
python main.py
